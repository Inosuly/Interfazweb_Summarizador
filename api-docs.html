Summarizar

Documentación de la API

Endpoint: /api/summarize

Método: POST

Parámetros: Archivo PDF

Respuesta: JSON con el resumen del artículo

Ejemplo de uso:

fetch('/api/summarize', {
    method: 'POST',
    body: formData
})
.then(response => response.json())
.then(data => console.log(data));
